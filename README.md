# Cube Timer

A feature-rich task management and timer application built with React, TypeScript, and Tailwind CSS.

## Features

- Task management with priorities, tags, and due dates
- Built-in timer with alarm system
- Customizable user profiles with avatar generator
- Responsive design

## Tech Stack

- React 18 + TypeScript
- Vite + SWC
- Tailwind CSS + shadcn/ui
- Framer Motion
- Jest + Testing Library
